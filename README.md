# Skill Anecdotes de France

This template is a French Fact Skill to demonstrate the usage of some basic features of the Alexa Skill Kit in French such as :

*Interaction Model*
* Intents
* Utterances
* Slots

*Backend*
* Request Handler
* Interceptors
* Session Attributes
* Persistent Attributes
